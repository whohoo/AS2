createBox

public function createBox(scaleX:Number, scaleY:Number, rotation:Number, tx:Number, ty:Number):Void

The createBox() method includes parameters for scaling, rotation, and translation. When applied to a matrix it sets the matrix's values based on those parameters. Using the createBox() lets you obtain the same matrix as you would if you were to apply the identity(), rotate(), scale(), and translate() methods in succession.
